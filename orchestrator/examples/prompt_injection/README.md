## Prompt Injection Example

This example is intentionally kept outside the core orchestrator package.

Run it from the repository root with:

```bash
python orchestrator/examples/prompt_injection/demo_launcher.py https://<your-ngrok-url>
```

Or from the `orchestrator/` directory with:

```bash
python examples/prompt_injection/demo_launcher.py https://<your-ngrok-url>
```
